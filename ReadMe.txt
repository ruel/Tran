========================================================================
    Tran 0.1
========================================================================

Tran is a program that cleans a specified directory. It looks for
archive files, and if it's already extracted it will be removed.
This is useful especially on cleaning download folders and removing
unused archive files.

/// Compilation and Testing

This is compiled, built, and tested on MSVC++ 2010

/// Usage

> Tran.exe [path]

/// Header Files

stdafx.h	->	General header file, contains all other include pre-processors
targetver.h	->	Default header file from MSVC++

/// Source Files

stdafx.cpp	->	Contains function definitions
Tran.cpp	->	Main source file containing the main function.
